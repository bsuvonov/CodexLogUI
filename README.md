# Codex Log UI

<img width="1922" height="1171" alt="image" src="https://github.com/user-attachments/assets/9b9b6557-e7a9-418f-a5c6-c6ce930f4193" />

Local web app for browsing Codex CLI session history.

It reads sessions from:

- `~/.codex/sessions/`
- `./sessions/` (inside this project, same file format as Codex sessions)

## Run

Requirements:

- Node.js 18+

Start the app:

```bash
npm start
```

Then open:

```text
http://127.0.0.1:4173
```
